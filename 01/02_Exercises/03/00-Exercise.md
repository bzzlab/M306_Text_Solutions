### Exercise 3: Setup ZAP Proxy on your Desktop OS  
###  Prerequisite
You have a running Desktop System (Windows, Mac, Linux)
3. You have downloaded ZAP Proxy from ```https://www.zaproxy.org/download/```

### Task 1: Update Java Runtime
1. Check Version of Java Runtime.
```bash
java --version
java 25.0.1 2025-10-21 LTS
Java(TM) SE Runtime Environment (build 25.0.1+8-LTS-27)
Java HotSpot(TM) 64-Bit Server VM (build 25.0.1+8-LTS-27, mixed mode, sharing)
```
2. Continue only if your version is lower than 17: Update to Version 25 (or at least to version 17). Download your appropriate of Java Runtime Edition. For Windows Desktop I recommend MSI-Version (https://www.oracle.com/java/technologies/downloads/#jdk25-windows)
3. Check again if the update Java runtime has been installed.
